#  Blinkit Sales Analysis Dashboard | Power BI

An interactive **Power BI dashboard** inspired by **Blinkit (India‚Äôs Last Minute App)** to analyze sales performance, outlet growth, item distribution, and customer ratings.  
This project highlights my ability to transform raw data into **meaningful business insights using Power BI**.

---

##  Project Overview

The **Blinkit Sales Dashboard** provides a consolidated view of business performance across multiple dimensions such as **outlet type, item category, outlet size, and location tiers**.  
It enables stakeholders to track trends, identify top-performing segments, and make data-driven decisions.

---

## Key KPIs

- **Total Sales:** 1.20M  
- **Total Ratings:** 33K  
- **Total Items:** 8,523  
- **Average Rating:** 3.92  

---

##  Dashboard Insights

-  **Outlet Growth Trend:** Sales variation based on outlet establishment year  
-  **Outlet Type Analysis:** Comparison of different outlet formats  
-  **Outlet Location Type:** Tier-wise sales contribution  
-  **Item Type Performance:** Top-selling item categories  
-  **Fat Content Analysis:** Sales split by product fat content  
-  **Outlet Size Distribution:** Small, Medium & High outlet contribution
